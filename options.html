TabBeacon

AIやWebアプリの「考え中 / 処理中」を、タブアイコンで見分けるための MVP 設定画面です。

方針

  • selectorType は auto / css / xpath に対応
  • auto は入力文字列から CSS / XPath を自動判定
  • Smart busy detection は aria-busy="true" や Stop 系ボタンを補助的に検出
  • 今は 1 条件 1 ルールの小さな構成です

ルール

入力例

CSS セレクタ

[aria-busy="true"] button[data-testid*="stop"]

XPath

//*[@aria-busy="true"] //button[contains(@aria-label, "Stop")]

URL パターン

https://chatgpt.com/* https://claude.ai/*
