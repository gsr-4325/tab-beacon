TabBeacon

AIやWebアプリの考え中 / 処理中を、DOM条件と network 条件の両方から見分けるための設定画面です。

方針

  • 1 ルールの中に複数条件を持てます
  • ANY はどれか 1 つ一致、ALL は全部一致です
  • 条件は DOM と Network を混在できます
  • Smart busy detection はルール全体に対する補助シグナルです

ルール

入力例

DOM 条件

[aria-busy="true"] //button[contains(@aria-label, "Stop")]

Network 条件

urlContains: /backend-api/ pathPrefix: /backend-api/conversation

URL パターン

https://chatgpt.com/* https://claude.ai/*
