Tab Beacon

Configure DOM and network conditions to show a busy state on the tab icon.

  • A single rule can have multiple conditions.
  • ANY matches when at least one condition is true. ALL matches when every condition is true.
  • You can mix DOM and network conditions in the same rule.
  • Smart busy detection acts as a rule-wide fallback signal.

Rules

Use this section to install a local sandbox preset without overwriting your own rules.

For real file:// testing, open manual-tests/tabbeacon-sandbox.html from your local repo.

Examples

DOM condition

[aria-busy="true"] //button[contains(@aria-label, "Stop")]

Network condition

urlContains: /backend-api/ pathPrefix: /backend-api/conversation

URL pattern

https://chatgpt.com/c/* https://chatgpt.com/g/*/c/*
