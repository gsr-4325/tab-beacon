Tab Beacon Sandbox

Tab Beacon の DOM 条件と network 条件を手動で試すためのローカルページです。

idle

1. DOM 条件テスト

おすすめ条件: [aria-busy="true"]

2. Network 条件テスト

最初は urlContains: postman-echo.com + GET + fetch/xhr をおすすめします。

注意: file:// からの network はブラウザ制約や CORS の影響を受けることがあります。

3. 参考ルール

URL pattern
file:///*manual-tests/*
chrome-extension://*/manual-tests/*
extension://*/manual-tests/*

Condition A
source: dom
selectorType: css
query: [aria-busy="true"]

Condition B
source: network
matchType: urlContains
value: postman-echo.com
method: GET
resourceKind: fetch/xhr

matchMode: ANY
